# Chief Template

A beginner-friendly template for the Roblox game framework Chief.

## Features

- **Simplified setup** — run `pesde run dev` to start building with darklua and serving with Rojo.
- **Snippets** for traits and providers.
- **Batteries included.**

## Getting Started

1. Make sure you have [pesde](https://pesde.dev/) and [Rojo](https://rojo.space/) installed.
2. Clone or use this template to start a new project.
3. Run the dev command:

   ```sh
   pesde run dev
   ```

   This builds the project with darklua and starts a Rojo server so you can sync straight into Roblox Studio.
